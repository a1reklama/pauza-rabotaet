# pauzarabotaet.ru WordPress MVP

В папке `wp-content/themes/pauza-rabotaet` лежит готовая WordPress-тема для MVP сайта:

- главная страница с маршрутом новичка;
- карта 12 шагов и отдельные страницы шагов;
- вкладки внутри каждого шага: обзор, условия, действия, полный текст, переход дальше;
- отдельная вкладка `Материалы` внутри шага для ссылок, упражнений, глоссариев и важных блоков;
- раздел спонсоров/проводников с фильтром по полу;
- раздел `Новости` для объявлений, обновлений групп и новых видео;
- страница калькулятора с местом для готового shortcode/HTML;
- раздел "Только сегодня";
- страница перехода в бот 4 шага;
- стартовый импорт контента из `pauzarabotaet.ru.docx`.

## Установка

1. Скопировать папку `wp-content/themes/pauza-rabotaet` в WordPress-проект или загрузить `pauza-rabotaet-theme.zip` через админку.
2. В админке WordPress открыть `Внешний вид -> Темы` и активировать `Pauza Rabotaet`.
3. После активации тема создаст стартовые страницы, 12 шагов, материалы, меню и настройки.
4. Открыть `Пауза работает` в левом меню админки и проверить глобальные ссылки.
5. Открыть `Спонсоры` в админке: импортированные контакты созданы как черновики. Публиковать только после подтверждения согласия.

## Где Редактировать

- `12 шагов`: номер, статус, цель, условия входа, чек-лист, Telegram/MAX-ссылки.
- `Полный текст шага`: длинный текст из документа, который виден только во вкладке `Полный текст`.
- `Материалы и ссылки шага`: видео, группы, диск, бот, внешние инструкции.
- `Упражнения и важные блоки`: глоссарии, молитвы, дефекты, списки, ВДА и другие блоки из DOCX.
- `Спонсоры`: имя, пол, телефон, Telegram/WhatsApp/MAX, примечание.
- `Новости`: объявления, новые видео, обновления ссылок групп, важные сообщения.
- `Только сегодня`: тексты основателя.
- `Материалы`: каналы, скачивание, инструкции, внешние инструменты.
- `Пауза работает`: глобальные ссылки, калькулятор, служебные тексты.

## Калькулятор

Готовый калькулятор не найден в исходной папке, поэтому тема оставляет управляемое место под него. Вставьте shortcode или HTML в поле:

`Пауза работает -> Калькулятор -> HTML/shortcode готового калькулятора`.

## Проверки Перед Запуском

- сверить перенос по контрольной карте: `docs/content-map.md`;
- открыть путь новичка: главная -> спонсоры -> 1 шаг -> калькулятор -> Telegram/MAX;
- проверить, что страница шага не перегружена: длинный текст открывается только отдельной вкладкой;
- опубликовать только подтвержденных спонсоров;
- проверить все внешние ссылки;
- проверить мобильную версию;
- обновить постоянные ссылки WordPress после активации темы, если архивы не открываются сразу.

## Важно Для REG.RU

Тема рассчитана на обычный WordPress-хостинг REG.RU с PHP и MySQL/MariaDB. Для боевого сайта не нужен VDS, Docker, Node или отдельный backend. Все CSS/JS уже лежат готовыми файлами внутри темы.

UX специально упрощен для людей с низкой технической грамотностью: крупные кнопки, короткие подписи, один следующий шаг на экране и скрытие длинных текстов во вкладках.
