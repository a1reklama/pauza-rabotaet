# Контент-карта DOCX -> сайт

Цель этой карты: ни один смысловой блок из `pauzarabotaet.ru.docx` не удаляется. UX может менять порядок и видимость материала, но каждому блоку назначается место на сайте.

## Основная Навигация

| Раздел сайта | Что хранит | WordPress-сущность |
| --- | --- | --- |
| Начать | простой маршрут новичка: спонсор -> 1 шаг -> калькулятор -> группа/бот | главная страница |
| Спонсоры | список контактов, фильтр по полу, правило “сначала написать” | `pauza_sponsor` |
| 12 шагов | карта шагов и страницы шагов с вкладками | `pauza_step` |
| Калькуляторы | внешние Telegram/MAX-боты, инструкция, куда отправлять результат | страница + настройки темы |
| Новости | объявления, новые видео, изменения ссылок, важные сообщения | `pauza_news` |
| Материалы | каналы проекта, группы шагов, боты, видео, калькуляторы, инструкции, скачивание | `pauza_material` |
| Только сегодня | авторские тексты владельца | `pauza_today` |
| Бот 4 шага | внешний инструмент, привязанный к 4 шагу | `pauza_material` + вкладка 4 шага |

## Что Добавляется Как Раздел, Хотя Не Расписано В DOCX

| Раздел | Почему нужен |
| --- | --- |
| Новости | нужен по продуктовой логике: объявления, новые видео, обновления групп, изменения ссылок. Не заменяет слово “Новости” из списка зависимостей первого шага. |

## Типы Материалов

| Тип | Что означает | Где показывать |
| --- | --- | --- |
| `project_channel` | общий канал проекта в Telegram/MAX или другой соцсети | `Материалы`, иногда главная |
| `step_group` | группа конкретного шага | вкладка `Видео и ссылки` нужного шага |
| `bot` | внешний бот-инструмент, привязанный к шагу | нужный шаг и `Материалы`, не футер |
| `video` | видео или плейлист | нужный шаг по дням/порядку, глобально только каналы |
| `calculator` | внешний калькулятор-бот | страница `Калькуляторы` |
| `download` | архив или файл | `Материалы`, при необходимости шаг |
| `instruction` | инструкция без личных ответов | шаг или `Материалы` |

## Правила Переноса

- Полный исходный текст каждого шага хранится и доступен во вкладке `Полный текст`.
- Краткий UX-слой не заменяет полный текст, а помогает человеку не потеряться.
- Внешние боты, группы и каналы не смешиваются: группа нужна для общения по шагу, канал для материалов проекта, бот для конкретного действия.
- Калькуляторы не встраиваются в сайт и не редактируются как shortcode. Это внешние Telegram/MAX-инструменты.
- Список спонсоров не публикуется автоматически без подтверждения согласия.
- Если в DOCX появляется новый блок, он сначала добавляется в эту карту, затем в сайт.
