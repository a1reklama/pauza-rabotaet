## Критичные потери

Критичных потерь текста в `Работа по шагу` и `Материалы шага` не найдено.

Проверенные рискованные места:

- Шаг 1, пункт 9: список зависимостей от `Порно` до `Трудоголизм` включен.
- Шаг 1, пункт 10: пять вопросов включены.
- Шаг 2, пункт 12: длинный текст про похоть включен через раскрытие.
- Шаг 3, пункт 14: упражнение на знание глоссария включено через раскрытие.
- Шаг 6, пункт 1: пять вопросов по дефекту включены.
- Шаг 8, пункт 14: упражнение ВДА включено через раскрытие.
- Шаг 9, пункт 4: график материальных возмещений включен.
- Шаг 11, пункты 2 и 3: просьбы и список вопросов включены.

## Что требует исправления

По результатам текущей проверки обязательных исправлений по потере материала нет.

Уже исправлено в ходе аудита:

1. WordPress-шаблон `single-pauza_step.php` теперь выводит внизу фактические строки перехода из `Текст руководителя`.
2. Для 12 шага вместо редакционного пояснения показываются финальные строки из текста руководителя.

## Допустимые редакционные замены

Эти изменения не считаются потерей материала:

- `проводник` → `спонсор`;
- `Первый/Второй/Третий шаг для ВСЕХ` → `1/2/3 шаг для ВСЕХ`;
- длинный URL в тексте заменен на кликабельное слово `Открыть`, если исходный адрес сохранен в `href`;
- длинные продолжения внутри задания скрыты под `Показать продолжение`, но остаются доступными пользователю.

## Автоматические проверки

- Все 12 шагов прочитаны из `step-full-texts.json`.
- Для каждого шага проверено покрытие номерных пунктов и строк-продолжений до следующего номера.
- Для каждого шага проверено, что все URL из текста руководителя присутствуют в материалах шага.
- Для шагов 1, 2, 3, 8 и 11 отдельно проверены длинные и вложенные фрагменты.
